What is git?

Defination
Git is a local version control system that tracks changes in your code and helps you go back in time if things break.

Analogy
Git is like a time machine for your code -- every save becomes a version you can revisit.

What is gitHub?

Defination
GitHub is an online platform to store and share your Git-tracked code.

Analogy
GitHub is like Google Drive for code -- it stores your work and lets you collaborate with others.
